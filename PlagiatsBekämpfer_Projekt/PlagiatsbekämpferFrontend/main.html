Um die Tests auf Plagiate zu überprüfen, haben wir die "Shingles" -Methode verwendet.

Mit Hilfe des Shingle Algorithmus kann erkannt werden, ob ein Text einzigartig ist. Eine Textpassage kann dann mit einer beliebigen anderen verglichen werden und auf Duplicate Content untersucht werden.

Shingles sind sich überschneidende Satzteile des Textes, die aus einer festgelegten Länge von Wörten bestehen.

Ein Beispiel ist: "Es wäre sehr schön, eine gute Note in Web Anwendung zu bekomemn"

Shingle 1 = Es wäre sehr

Shingle 2 = wäre sehr schön

Shingle 3 = sehr schön eine

Shingle 4 = schön eine gute
